## Building P3icli 
p3icli is a Win32 console application built primarily with Visual Studio.

In addition, several [cygwin tools](https://cygwin.com) are invoked by VS
during the build.

Build instuctions are enumerated in detail in the file
doc/building_p3icli_from_sources.html, which can be
[browsed here](https://htmlpreview.github.io/?https://github.com/decaff/p3icli/blob/master/doc/building_p3icli_from_sources.html).
